What is HTML and how does it work?

HTML consists of a series of short codes typed into a text-file by the site author. HTML stands for Hyper Text Markup Language and it is used for creating a web page. HTML gives a structure to the web page and it's content. HTML has elements, tags, lists, attributes. which is used for web page designing.
